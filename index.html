Comandos Descripcion
git status Nos da el estado actual de nuestros archivos del repositor
git add . Prepara los archivos (etapa staged) para ser comitiados (confirmar cambios)
git commit -m "mensaje de commit" Confirma los cambios que estaban en preparacion (es staged)
git commit -am "mensaje de commit" Agrega y confirma los cambios que estaban en preparacion (es staged) en un solo comando
git commit --soft HEAD~1 Deshace el ultimo commit de mi repo local (no publicado) manteniendo los cambios
git commit --amend -m "Este es el mensaje correcto" Arregla el mensaje que usaste en el ultimo commit
git branch nombreRama Crea una rama nueva (hay que colocar nombre (ej: development=desarrolo))
git checkout nombreRama Salto a la rama que quiero (colocar el nombre de dicha rama)
git merge nombreRama Une dos ramas
git checkout -b nombreRama Crea y salta a una nueva rama
git remote -v Me da info de los repositorios remotos vinculados con mi repo local
git push origin nombreRama Sube cambios a mi repo remoto
git pull origin nombreRama Baja la rama remota con sus cambios a mi repo local
